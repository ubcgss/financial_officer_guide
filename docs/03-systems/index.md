# Systems

The FO interacts with the following systems. Each page below covers the
system's purpose, FO-specific access, and key workflows.

| System | Purpose | FO access level |
|--------|---------|----------------|
| [Zoho Expenses](zoho/) | Expense approval workflow | Final approver (level 4) |
| [Zoho Books](zoho/) | Budget and actuals, financial statements | Read / request access |
| [Banking](banking/) | Scotiabank, Vancity (in transition), Plooto | Full debit card + online access |
| [Payworks](banking/) | Timesheets, semi-monthly payroll | Approval of WL timesheets |
| [Investments](investment/) | GICs, Mutual Funds, RBC DS (in transition) | Review / signatory |
