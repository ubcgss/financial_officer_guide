# Zoho Expense Guide

## Bulk Updating a Report's Project
If an expense belongs to a specific project (e.g., Meal Subsidy or Baby Hamper), make sure to assign it to that project so costs can be tracked accurately. To update multiple expenses at once, select all relevant expenses, then choose `Bulk Update`:

<img src="https://github.com/ubcgss/transition_guide/blob/main/images/zoho_projects/zoho_projects_1.png" align="center" height="60%" width="60%"> 

In the `Bulk Field Update` menu, choose the correct `Customer` (the project name) and then the matching `Project` (academic/fiscal year included):

<img src="https://github.com/ubcgss/transition_guide/blob/main/images/zoho_projects/zoho_projects_2.png" align="center" height="60%" width="60%">
