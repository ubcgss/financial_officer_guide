# Transition Guide

## Set Up Vancity Credit Card Online Access
You can set up the login details for the Vancity credit card on [creditcards.vancity.com](creditcards.vancity.com). Setting this up is necessary to view the credit card statements and account balance. 

The user for your Vancity credit card is different from the banking one. If you have any issues with the setup, you can call Vancity Card Services at 1-800-611-8472.
